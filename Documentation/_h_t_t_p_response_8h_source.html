00109  *  - Wait for asyncSocket to notify it that the data has been sent.

00116  * This allows the response to asynchronously execute any code needed to calculate a part of the header.

00118  * or perhaps the response needs to look for a resource on network-attached storage.

00121  * After returning YES from this method, the HTTPConnection will wait until the response invokes its
00122  * responseHasAvailableData method. After this occurs, the HTTPConnection will again query the delayResponseHeaders
00123  * method to see if the response is ready to send the headers.
00124  * This cycle will continue until the delayResponseHeaders method returns NO.

00127  * Asynchronously generating the body of the response is not an excuse to delay sending the headers.
