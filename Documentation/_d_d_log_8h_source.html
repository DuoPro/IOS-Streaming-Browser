00038 // This question is actually composed of two parts:
00039 // 1. Is it available to the compiler?
00040 // 2. Is it available to the runtime?

00042 // For example, if we are building a universal iPad/iPhone app,
00043 // our base SDK may be iOS 4, but our deployment target would be iOS 3.2.
