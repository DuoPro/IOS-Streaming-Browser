00587 // The readData and writeData methods won't block (they are asynchronous).
00588 // 
00589 // When a read is complete the socket:didReadData:withTag: delegate method is dispatched on the delegateQueue.
00590 // When a write is complete the socket:didWriteDataWithTag: delegate method is dispatched on the delegateQueue.
00591 // 
00592 // You may optionally set a timeout for any read/write operation. (To not timeout, use a negative time interval.)
00593 // If a read/write opertion times out, the corresponding "socket:shouldTimeout..." delegate method
00594 // is called to optionally allow you to extend the timeout.
00595 // Upon a timeout, the "socket:willDisconnectWithError:" method is called, followed by "socketDidDisconnect".
00596 // 
00597 // The tag is for your convenience.
00598 // You can use it as an array index, step number, state id, pointer, etc.
